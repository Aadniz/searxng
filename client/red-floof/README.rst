=====================
MEMO vite development
=====================

Local install::

  # in folder ./client/red-floof/
  $ npm install

Start development server::

  $ ./manage vite.red-floof.dev

  # in folder ./client/red-floof/
  $ npm exec -- vite

Fix source code::

  # in folder ./client/red-floof/
  $ npm run fix

Fix & Build::

  $ ./manage vite.red-floof.build
